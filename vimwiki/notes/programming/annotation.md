## 文件注释

```
/*
 * @author: 作者
 * @date: 时间
 * @description: 描述
 */
```

```
/*
 * 文件说明
 *
 * 版本号
 * 
 * @category 类型
 * @package  包
 * @author   作者
 * @license  许可
 * @link     链接
 */
```

```

```

## 方法注释

```

 * 方法说明
 * 
 * @param 参数类型 参数名字 含义
 * 
 * @return 返回值类型
 */
```
